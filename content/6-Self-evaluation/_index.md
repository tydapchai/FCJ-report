## Internship Overview

Throughout my internship at **First Cloud Journey (FCJ)** from **September 6, 2025** to **December 24, 2025**, I was immersed in the AWS cloud ecosystem, contributed to a collaborative team project, and gained valuable exposure to professional workflows and industry practices.

---

## Key Accomplishments

The internship program was structured around two complementary tracks that ran concurrently:

### AWS Learning Journey

- Completed **6 comprehensive AWS Modules** spanning core services including networking, compute, storage, security, and database technologies
- Successfully finished **33 hands-on laboratory exercises** ranging from foundational EC2 configurations to advanced implementations involving CDK, SageMaker, and CloudWatch monitoring
- Participated in **6 AWS community events** including Vietnam Cloud Day, GenAI Builder Club, AWS Cloud Mastery Series (#1, #2, #3), and Building Agentic AI workshop

### Team Project (MapVibe)

- Collaborated with a **5-member development team** to create a restaurant discovery and search application
- Architected the **database layer** utilizing PostgreSQL with PostGIS spatial extensions and pgvector for embedding storage
- Engineered a **hybrid search system** integrating BM25 lexical search with semantic vector similarity matching
- Implemented a **RAG (Retrieval-Augmented Generation) pipeline** enabling natural language query processing
- Orchestrated the **complete infrastructure migration** to AWS using Terraform for Infrastructure as Code

This dual-track experience enhanced my capabilities in cloud architecture design, Infrastructure as Code practices, serverless application development, and effective collaboration within a team environment with real deliverables and timelines.
