# Summary Report: "Building Agentic AI - Context Optimization with Amazon Bedrock"

## Event Overview

| | |
|---|---|
| **Event Name** | Building Agentic AI - Context Optimization with Amazon Bedrock |
| **Date** | Friday, December 5, 2025 |
| **Role** | Attendee |
| **Format** | Hands-on workshop |
| **Location** | 26th Floor, Bitexco Financial Tower, District 1, Ho Chi Minh City, Vietnam |
| **Main Topic** | Developing autonomous AI agents using Amazon Bedrock through practical techniques and industry use cases |

This immersive workshop concentrated on constructing autonomous AI agents leveraging Amazon Bedrock capabilities. The event assembled industry experts from AWS, Azure, and the Kubernetes ecosystem to deliver insights on agentic AI development, context optimization methodologies, and production-ready implementation approaches. Participants gained hands-on experience with AWS Bedrock Agent Core, explored practical applications, and engaged in building agentic workflows during the workshop session.

## Session Summaries

### Session 1: AWS Bedrock Agent Core (9:10 - 9:40)
**Presenter:** Kien Nguyen

Kien Nguyen delivered a comprehensive presentation on the AWS Bedrock Agent Core architecture and its core capabilities. The session provided an in-depth exploration of Agent Core components and the runtime execution environment.

**Topics Discussed:**
- Amazon Bedrock Agent architectural foundations
- Agent Core components and execution runtime
- Tool integration patterns and action group configuration
- Memory management and state persistence for agents
- Best practices for agent development lifecycle

---

### Session 2: Implementing Agentic Workflows on AWS (9:40 - 10:00)
**Presenter:** Viet Pham

Viet Pham presented real-world implementations of agentic AI solutions, demonstrating how organizations are leveraging these technologies in production environments.

**Key Highlights:**
- Production-grade applications of autonomous AI agents
- Workflow orchestration design patterns
- Enterprise system integration strategies
- Lessons learned from real-world deployments

---

### Session 3: CloudThinker Platform Overview (10:00 - 10:10)
**Presenter:** Thang Ton

Thang Ton introduced the CloudThinker platform and outlined their AI solution offerings, setting the stage for the technical deep dive that followed.

---

### Session 4: Agentic Orchestration & Context Optimization (10:10 - 10:40)
**Presenter:** Henry Bui

Henry Bui delivered an L300-level technical deep dive on CloudThinker's Agentic Orchestration framework and context optimization strategies for Amazon Bedrock.

**Technical Topics:**
- Context window optimization techniques and strategies
- Agentic orchestration patterns and frameworks
- Advanced methodologies for enhancing agent performance
- Production deployment considerations and best practices
- Cost optimization through efficient context management

---

### Session 5: Hands-on Workshop (11:00 - 12:00)
**Facilitator:** Kha Van

Kha Van led an interactive hands-on workshop where participants built agentic AI applications using the CloudThinker platform and Amazon Bedrock.

**Workshop Activities:**
- Constructing agents with Amazon Bedrock
- Implementing context optimization techniques in practice
- Testing, debugging, and iterating on agent workflows
- Collaborative problem-solving and knowledge sharing

## Personal Reflections

This workshop delivered valuable perspectives on the rapidly advancing field of agentic AI. The blend of theoretical presentations from AWS specialists and practical exercises with CloudThinker's tooling provided a holistic understanding of autonomous AI agent development.

The venue at Bitexco Financial Tower facilitated excellent networking opportunities with professionals spanning AWS, Azure, and Kubernetes communities. These cross-platform discussions enriched the learning experience and provided diverse perspectives on AI agent implementation.

The L300-level session on context optimization proved particularly beneficial, offering advanced techniques immediately applicable to improving agent performance in production settings. The hands-on workshop enabled attendees to apply these concepts in real-time, cementing the theoretical knowledge through practical application.

### Key Takeaways
- Agentic AI represents a significant evolution in how we build and deploy AI systems
- Context optimization is critical for both performance and cost management
- The AWS Bedrock ecosystem provides robust tooling for enterprise-grade agent development
- Cross-functional collaboration between different cloud communities drives innovation
